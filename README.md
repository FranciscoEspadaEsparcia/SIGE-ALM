# SIGE-ALM – Sistema Integral de Gestión de Almacén

**Autor:** Francisco Espada Esparcia  
**Asignatura:** 47GIIN – Proyectos de Ingeniería del Software (VIU)

## 📘 Descripción
SIGE-ALM es un sistema para la gestión de almacenes: artículos, movimientos de stock, órdenes de trabajo, alertas y usuarios.  
El desarrollo se gestiona con metodología **Scrum**, mediante épicas, historias de usuario y sprints.

## 🗂️ Estructura del repositorio
- **/Documentación** → Entregables: Product Backlog, Planificación Sprint 1, diagramas.
- **/Diseño** → Diagramas, mockups y artefactos de análisis/diseño.
- **/Código** → Se añadirá en la fase de implementación.

## 🗓️ Planificación (resumen)
- **Sprint 1:** Núcleo funcional (artículos, movimientos, OT, usuarios).  
- **Sprint 2:** Reportes, alertas y configuración.  
- **Sprint 3:** Proveedores, pruebas finales, correcciones y documentación.
